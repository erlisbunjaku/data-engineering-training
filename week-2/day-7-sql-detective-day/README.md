# Day 7 - SQL Detective Day

## Goal of the practice

The goal of this practice was to improve my SQL skills by fixing broken queries, creating validation queries, and making a business report using real data.

## Files included

- broken_queries.sql - Queries with errors.
- fixed_queries.sql - Fixed versions of the broken queries.
- validation_queries.sql - Queries used to check and verify the data.
- verified_business_report.md - Report based on the SQL results.
- query_explanations.md - Short explanations of queries.

## How to run the SQL files

Run the files in this order:

1. broken_queries.sql
2. fixed_queries.sql
3. validation_queries.sql
4. verified_business_report.md

## What I learned about debugging SQL

I learned how to find and fix SQL errors. I had some struggles with JOINs and understanding the ON condition, but I am working on improving that part.

## What I learned about verifying business reports

I learned that reports should be based on SQL results and every important number should have a query that proves it.
