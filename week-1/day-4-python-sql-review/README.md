# Day 4 - Python and SQL Logic Review

## Practice Goal

This practice focused on reviewing Python and SQL data processing logic. 
The goal was to work with order data, filter records, calculate values, sort data, and understand the similarities and differences between Python and SQL.

## Files Included

- `python_orders.py` - Python tasks using lists, dictionaries, loops, functions, filtering, sorting, and calculations.
- `setup.sql` - Creates the orders table and inserts sample order data.
- `sql_queries.sql` - SQL queries for selecting, filtering, sorting, and calculating data.
- `comparison.md` - Comparison between Python and SQL approaches.
- `daily_report.txt` - Short notes and business challenge answers.
